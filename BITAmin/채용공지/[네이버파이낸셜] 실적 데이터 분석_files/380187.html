실적 데이터 분석

업무 내용

1. 실적관리 및 실적집계 기반시스템 운영

- Data Analysis : 네이버페이 결제사업 실적의 실측치와 예측치 관리

- Data Preprocessing : 데이터 재처리/가공

- Data Pipeline 유지보수 : 데이터 집계용 배치, 메일발송 서비스 관리

- Data Quality 보장 : 결제데이터 정합성 검증, 원천테이블 변화감지/보완 (Anomaly Detection)

- Data Visualization 유지보수 : KPI 관리용 대시보드에 수정사항 반영

2. 실적예측 고도화

- 시계열 모형 예측 오차 범위 축소

- 예측 타겟 지표 확대 및 선행 지표 탐색

(사업적 의사결정에 예측치가 적극 활용되고 있어 다른 구성원이 예측치를 쉽게 이해 / 

 활용할 수 있는 형태로 설명하고 가공하는 역량이 필요합니다. )


자격 요건

- Hiive Query or SQL 숙련자

- Python 숙련자

- Python 활용 시계열 예측 모형 생성/보완 경험자


우대사항

- 동종/유사업계 실적관리 경험자

- 데이터 배치작업 생성/운영 경험자

- raw data 탐색의 다양한 시행착오 경험자
