Chapter4 판단 예측

-핵심 원리
예측 작업을 간결하고 분명하게 정하기 -> 애매한 용어x 감정적 용어 x
체계적인 접근 방식을 구현하기 -> 요인, 가격, 품질, 수요 등 
기록하고 정당화하기 -> 문서형태로
예측값을 체계적으로 평가하기 -> 기록, 피드백
예측가와 사용자를 구분하기

- 델파이 기법
1.전문가 위원회가 모입니다. (익명성)
2.예측 작업과 과제를 정하고 각 전문가에게 나눠줍니다.
3.각 전문가는 초기 예측값과 타당한 이유를 돌려줍니다. 피드백을 제공하기 위해 모아서 요약합니다.
4.각 전문가에게 피드백을 제공합니다. 각 전문가는 피드백에 비춰보아 예측값을 검토합니다. 이 과정은 만족스러운 수준으로 합의가 이뤄질 때까지 반복될 수도 있습니다.
5.각 전문가의 예측값을 모아서 최종 예측값을 구성합니다.

-> 피드백은 요약 통계, 시각화

- 유사점으로 예측하기
주택 감정과정으로 가격을 결정(토지, 주거 면적, 침실과 욕실의 개수, 차고 공간 등)

- 시나리오 예측
모든 가능한 요인과 동인들, 그것의 상대적인 영향, 상호작용, 예측할 목표 등을 고려하여 시나리오 생성

- 신제품 예측
1. 판매원 의견 종합법
-> 아울렛/지점/매장 예측값 및 수요
2. 임원 의견
-> 델파이 접근 방식의 변형 형태 적용(추정-대화-추정)
3. 고객의 의도
-> 설문조사(구매의도, 구매행동사이 관계)

- 판단 조정
과거 데이터 사용 가능할 때 -> 프로모션, 대규모 스포츠 경기, 공휴일
-> 편향과 한계점이 따른다.
판단조정은 유의미한 추가 정보를 가지고 있거나 조정이 필요하다는 강력한 증거있을 때 효과적
구조화된 체계적인 접근 방식을 사용하면 판단조정의 정확도가 높아진다.
