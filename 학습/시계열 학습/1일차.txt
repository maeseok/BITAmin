-> 기법의 특성, 정확도, 비용 등을 반드시 고려

정량적인 예측 : 과거 수치 정보를 사용할 수 있을 때, 과거 패턴이 미래에도 계속될 때
-> 시계열 or 횡단면 데이터 사용 
분해 모델, 지수 평활, ARIMA 모델, VAR, VECM 등

추세, 계절성

예측의 단계
1. 문제 정의 : 방식, 요구, 예측 함수
2. 정보 수집 : 통계 데이터
3. 예비 분석 : 추세, 계절성, 경기 순환
4. 모델 선택 : 예측 변수, 설명 변수 사이 관계의 강도 등 
-> 회귀, 지수 평활, ARIMA, 동적 회귀, 계층적 예측, 신경망 벡터 자기회귀
5. 모델 사용 및 평가

- 통계적 예측 관점
확률변수, 예측구간(95% 등) -> 구간 예측, 점 예측 - 예측 분포


chapter2 시계열 시각화

계절성 - 시간별, 일별, 주별, 연간 등
-> 다중 계절성은 빈도 목록에서 하나를 고르지 않음

시계열 패턴
- 추세 : 데이터가 장기적으로 증가하거나 감소할 때 추세 존재
- 계절성 : 해마다 어떤 특정한 때나 1주일마다와 같은 계절성 요인, 빈도의 형태로 나타남
- 주기성 : 고정된 빈도가 아닌 형태로 증가나 감소하는 모습을 보일 때 주기 발생
-> 주기는 일정한 빈도로 나타나지 않는 요동, 계절성은 빈도가 변하지 않고 어떤 시기와 연관
-> 일반적으로 주기들의 평균 길이는 계절성 패턴의 길이보다 길고, 크기는 좀 더 변동성이 큼

계절성 그래프 : ggseasonplot - 계절성 패턴 강조
계절성 부시계열 그래프 : ggsubseriesplot - 계절에 대한 데이터를 모아 분리된 작은 시간 그래프
-> 계절성이 시간에 따라 어떻게 변하는지
산점도 : qplot - 변수 사이의 관계를 시각화하는데 도움(상관계수)
시차 그래프 : gglagplot - 분기별

자기상관 : 시계열의 시차 값 사이의 선형 관계 측정 (계절성 시차에서 크게 나타남)
-> ACF(자기상관함수)를 나타내기 위해 그림

백색잡음 : 자기상관이 없는 시계열 (자기상관 거의 0)
